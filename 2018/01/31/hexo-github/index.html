hexo + github 搭建免费个人博客

发表于 2018-01-31

什么是Hexo？

Hexo是一款基于nodejs开发的轻量级博客，不需要数据库就可以直接跑起来的博客。

Hexo有什么优势？

除了上述的不需要数据库、webserver以外，Hexo还有以下优点：

  1. 编译后直接生成静态文件，可以直接访问。
  2. 可以和github配合构建免费博客。
  3. 文档编写使用md格式，更加方便。
  4. 有很多开源主题，可以根据喜好进行选择和替换。
